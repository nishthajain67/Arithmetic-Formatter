## What It Does

The program takes a list of arithmetic problems and arranges them vertically, side by side:

**Input:**
```python
["32 + 698", "3801 - 2", "45 + 43", "123 + 49"]
```

**Output:**
```
   32      3801      45      123
+ 698    -    2    + 43    + 49
-----    ------    ----    ----
```
